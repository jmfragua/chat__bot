# chat__bot

Chatbot inteligente para FAQs de **nómina** y **contratación** con interfaz mejorada en Streamlit.

## Funcionalidades

- Chat conversacional con `st.chat_message` y `st.chat_input`
- Respuestas automáticas por coincidencia inteligente de preguntas frecuentes
- Métricas rápidas en la cabecera
- Panel lateral con preguntas sugeridas para uso rápido

## Ejecución local

```bash
pip install -r requirements.txt
streamlit run app.py
```

Luego abre en tu navegador la URL local mostrada por Streamlit.
